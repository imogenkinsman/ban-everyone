# BanEveryone

Ban Everyone is a utility gem for scraping League of Legends Tribunal data.

## Installation

Add this line to your application's Gemfile:

    gem 'ban_everyone'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ban_everyone
